# React

## Quadrant

## Type

## Description
A JavaScript library for building user interfaces

``` js
import React from 'react';
import ReactDOM from 'react-dom'

export default class App extends React.Component {
  render() {
    return (
      <div>Hello React!</div>
    );
  }
}

ReactDOM.render(<App/>,document.getElementById('root'))
```
